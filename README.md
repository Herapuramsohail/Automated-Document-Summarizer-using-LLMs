# 🧠 Automated Document Summarizer using LLMs

An AI-powered document intelligence platform that lets you upload any document and instantly extract insights through multiple summarization modes and an interactive Q&A interface — powered by **Google Gemini**, **FastAPI**, **React + Vite**, and **FAISS**.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📄 **Document Upload** | Upload PDF, DOCX, or TXT files via drag-and-drop or browse |
| 🤖 **Abstractive Summarization** | Natural language summaries generated by Google Gemini LLM |
| 🔍 **Extractive Summarization** | Key sentence extraction in 4 formats: Short, Bullet Points, Executive Brief, Detailed |
| 💬 **Document Q&A (RAG)** | Ask questions about your document — grounded answers via Retrieval-Augmented Generation |
| 📊 **Metrics & Analytics** | Real-time ROUGE scores, readability, latency tracking, and session analytics |
| 🌑 **True Black UI** | Premium dark theme with glassmorphism, micro-animations, and indigo accents |

---

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | React 18, Vite, Material UI, TailwindCSS, Chart.js |
| **Backend** | FastAPI, Python 3.12, SQLAlchemy (SQLite) |
| **AI / LLM** | Google Gemini API (`gemini-1.5-flash`) |

## 🚀 Quick Start

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (recommended)
- **OR** Python 3.12+ and Node.js 20+
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

---

### Option A — Docker Compose (Recommended)

```bash
# 1. Clone the repo
git clone https://github.com/Herapuramsohail/Automated-Document-Summarizer-using-LLMs.git
cd Automated-Document-Summarizer-using-LLMs

# 2. Create your .env file from the example
cp .env.example .env
# Edit .env and set your GEMINI_API_KEY

# 3. Build and run
docker compose up --build

# App is live at: http://localhost
# API docs at:   http://localhost:8000/docs
```

---

### Option B — Local Development (Node.js + Python)

#### Backend

```bash
cd backend

# Create and activate virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS / Linux

# Install dependencies
pip install -r requirements.txt

# Create .env (copy from root .env.example, set your GEMINI_API_KEY)
copy ..\.env.example .env    # Windows
# cp ../.env.example .env    # macOS / Linux

# Run the FastAPI server
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

#### Frontend

```bash
cd frontend

# Install Node dependencies
npm install

# Run the Vite dev server
npm run dev

# App: http://localhost:3000
```

## 🔧 Environment Variables

Copy `.env.example` to `.env` and fill in:

```env

DATABASE_URL=sqlite:///./document_summarizer.db
UPLOAD_DIR=./uploads
VECTOR_DB_DIR=./vector_indices
```

Get a free API key at: **[https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)**

## 🌐 API Reference

Once running, visit **[http://localhost:8000/docs](http://localhost:8000/docs)** for the interactive Swagger UI.

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/v1/documents/upload` | Upload a document |
| `GET` | `/api/v1/documents/` | List all documents |
| `POST` | `/api/v1/summaries/generate` | Generate a summary |
| `GET` | `/api/v1/summaries/document/{id}` | Get summaries for a document |
| `POST` | `/api/v1/qa/document/{id}/query` | Ask a question (RAG) |
| `GET` | `/api/v1/metrics/analytics` | Get analytics data |
| `GET` | `/health` | Health check |

## ☁️ Deployment

### Deploy on any VPS / Cloud VM

```bash
# Install Docker on your server (Ubuntu example)
curl -fsSL https://get.docker.com | sh

# Clone and configure
git clone https://github.com/Herapuramsohail/Automated-Document-Summarizer-using-LLMs.git
cd Automated-Document-Summarizer-using-LLMs
cp .env.example .env
nano .env  # Set GEMINI_API_KEY

# Start in detached mode
docker compose up -d --build
```

The app will be available on **port 80** (frontend) and **port 8000** (backend API).
