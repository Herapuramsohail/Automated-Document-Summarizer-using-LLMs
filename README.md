# 🧠 Automated Document Summarizer using Large Language Models (LLMs)

An end-to-end **AI-powered Document Intelligence Platform** that leverages **Google Gemini**, **Retrieval-Augmented Generation (RAG)**, and **FAISS Vector Search** to generate intelligent document summaries, extract key insights, perform topic analysis, and answer context-aware questions from uploaded documents.

The platform supports **PDF, DOCX, and TXT** documents, offering multiple summarization modes—including **Abstractive**, **Extractive**, **Bullet Points**, **Executive Brief**, and **Detailed Summaries**—through a modern **React + Vite** frontend and a high-performance **FastAPI** backend.

## ✨ Key Features

* 📄 Upload and analyze **PDF, DOCX, and TXT** documents
* 🤖 AI-powered abstractive summarization using **Google Gemini**
* 🔍 Multiple extractive summarization formats
* 💬 Context-aware Document Q&A using **RAG + FAISS**
* 🏷️ Keyword extraction and topic analysis
* 📊 Real-time analytics, ROUGE evaluation, readability metrics, and latency tracking
* 🎨 Responsive React interface with Material UI and TailwindCSS
* ⚡ FastAPI backend with RESTful APIs
* 🐳 Dockerized deployment for scalable production environments

---

## 🛠️ Technology Stack

**Frontend:** React 18, Vite, Material UI, TailwindCSS, Chart.js

**Backend:** FastAPI, Python 3.12, SQLAlchemy, SQLite

**AI & NLP:** Google Gemini API, LangChain, FAISS, Google Embeddings, NLTK, Scikit-learn, ROUGE

**Deployment:** Docker, Docker Compose, Nginx, Vercel
